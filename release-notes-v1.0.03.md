# ParallelVrGallery Pro v1.0.03

- 视频左右眼从生成阶段直接分离，编码器分别绘制左右半屏，移除大 SBS 合成后再次切分的重复内存操作。
- 视频左右眼缓存改为并行 WebP Lossless 写入，压缩努力由 100 调整为 40；缓存仍为逐像素无损。
- 恢复任务优先读取首帧缓存，减少重复推理和全分辨率 Bitmap 拷贝。
- 视频调试指标改为吞吐、延迟、队列等待、缓存命中读取及完整队列深度。
- AVC 输出优先启用设备支持的 VBR/High Profile，并提高质量码率下限；旧 `encoderV12` 成品继续可浏览，已完成帧可供 V13 续跑。
